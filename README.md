# Goldstonian Concordance Bible Agents

This repository defines the **agent interaction layer** for the **Goldstonian Concordance Bible (GCB)** ecosystem.

Its purpose is to specify how AI agents, retrieval systems, theological assistants, and downstream tools should interact with:

- the canonical index
- the knowledge graph
- the ontology
- the doctrine framework
- the public scripture corpus

The GCB agent layer is built around the **81-Book Ethiopian Canon** and the **Mirror → Water → Fire Doctrine**.

---

# Core Purpose

This repository helps AI systems answer biblical, theological, and canon-structure questions using the GCB stack in a consistent and canon-aware way.

It defines:

- agent roles
- retrieval behavior
- canon resolution rules
- doctrine resolution rules
- distinction safeguards
- routing policies
